


# コピペ実行用

```bash
python -m venv .venv
source .venv/bin/activate

# pipを最新にする 
pip install --upgrade pip

# venv上の環境を整える
pip install -r requirements.txt

# chromedriverを実行可能に install済みだよな？
chmod +x /usr/local/bin/chromedriver

# main.pyを走らせる
python main.py
```
